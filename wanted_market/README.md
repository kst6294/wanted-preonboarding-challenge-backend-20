## 현재 상태
**미완성 프로젝트**
이 프로젝트는 아직 완료되지 않았으며, 일부 기능이 구현되지 않았습니다.

## 사용한 기술 스택
- **프로그래밍 언어**: Python
- **웹 프레임워크**: Django, Django Rest Framework (DRF)
- **데이터베이스**: MySQL
- **인증**: JWT (JSON Web Token) - djangorestframework-simplejwt
- **API 문서화**: drf-yasg (Swagger UI)
- **패키지 관리**: pip
- **기타**: Django 관리자 인터페이스
